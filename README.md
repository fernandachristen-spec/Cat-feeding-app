# Lügt er? - Cat Feeding Tracker

A simple app to track when Phil and Boris get fed.

## How to use
- Tap each time slot (07:00, 12:00, 18:00)
- Status changes: nein → ja → immer → nein
- Data resets daily at midnight

## Features
- Three feeding times per day
- Two cats tracked
- Automatic daily reset
